Hangman
========

A small game of hangman on the command line.

```
  Hangman
 =========
     _______
    |/     |
    |      O    GAME OVER :(
    |     /|)
    |     / )
 __/|\_________

 H A _ G M A _

 History: A, B, C, D, E, F, G, H, I, J, K, L, M, Z (10/10)
```

Usage:
```
$ python hangman.py <word>
```
